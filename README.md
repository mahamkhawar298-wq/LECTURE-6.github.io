# Bano Qabil | Digital Skills Empowerment Platform

A comprehensive educational platform for free IT courses, test registration, result checking, and student support.

## Features

- 4 core IT courses (Web Dev, Python/AI, Freelancing, Mobile Apps)
- Student registration with test date selection (localStorage)
- Result portal with roll number + mobile verification
- Contact inquiry system with validation
- Hafiz Naeem vision section
- Responsive gallery with campus photos
- Student reviews section

## Tech Stack

HTML · CSS · JavaScript · Font Awesome · localStorage

## Quick Start

Open `index.html` in any modern browser.
